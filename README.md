## ▍使用套件
* requests==2.25.1
* python = "3.7.1"
* pandas = "^1.2.2"
* bs4 = "^0.0.1"

## ▍Run crawler
Create your virtual environment
```
$ python3 -m venv venv
```
And enable virtual environment
```
$ . venv/bin/activate
```
Install requirements
```
$ pip install -r requirements.txt 
```
Run crawler
```
$ python3 main.py
```
